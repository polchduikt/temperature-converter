# Temperature Converter 🌡️

A simple Android application for converting temperatures between Celsius and Fahrenheit.

## Screenshot

![Screenshot](docs/screenshots/screenshot.jpg)

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Input validation
- Clean dark UI

## Tech Stack

- **Language:** Java
- **Platform:** Android
- **Min SDK:** 24
- **Layout:** LinearLayout

## Formula

°C → °F : F = C × 9/5 + 32

°F → °C : C = (F − 32) × 5/9
